Tutorial 01 - Using upROOT¶

Basic upROOT usage¶

First, we need to import the upROOT package:

In [1]:
import uproot

Important!
Make sure that you have upROOT version 3.4.15 - they recently added the support needed to read all data types that are used in our PicoDst format

In [170]:
uproot.__version__
Out[170]:
'3.4.15'

If you dont have the most updated version use:

pip install uproot --update

or

conda update uproot

depending on whether you installed it with pip or conda

upROOT makes it very easy to read data from ROOT data files. We can open a single file easily using the open method:

In [171]:
f = uproot.open( "/Users/jdb/bnl/work/MLatSTAR/st_physics_20057002_raw_7500003.picoDst.root" )
f.keys()
Out[171]:
[b'PicoDst;1']

upROOT documentation describes all of the methods that we use and their additional options: https://github.com/scikit-hep/uproot

After opening the file we can see whats in it by using the keys method. In this PicoDst file, we have only one key, the PicoDst TTree. We can get the PicoDst out and begin looking at the data:

In [172]:
PicoDst = f["PicoDst"]
PicoDst.show()

The show member shows us what the PicoDst format includes. The 2nd and 3rd columns include information about how the informantion is read out of the ROOT TTree. Most of the time the default works perfectly, but if needed you can override how the data is read to support complex/custom data types.

We can also use PicoDst.allkeys() if we just want the name of the keys

As you can see, the same basic procedure can be used to plot any TH1 type histogram, se we can define a helper function to do this. You can use something like this and add your own options for common settings:

In [62]:
def plotTH1( h, **kwargs ) :

    # Make a figure, use the default size if the opt 'fs' is not given
    fs = kwargs.get( "fs", (16, 9) )
    plt.figure(figsize=fs)

    x = [ x for x in h.edges[:-1] ]    
    plt.hist( x, weights=h.values, bins=h.edges, fill=True, histtype="step" );
    gca = plt.gca()
    
    # Activate logy by adding a keyword argument logy=True
    if kwargs.get( 'logy', False ) : gca.set_yscale('log')

    # Set the axis limits if given as keyword opts
    if 'ylim' in kwargs : gca.set_ylim( kwargs.get( 'ylim' ) )
    if 'xlim' in kwargs : gca.set_xlim( kwargs.get( 'xlim' ) )

    plt.show()
